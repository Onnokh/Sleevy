# Rybbit for Server-Side Product Analytics

Sleevy will capture product usage as server-side custom events sent from the API to a self-hosted Rybbit instance, treating the REST API as the single instrumentation point so that captures, opens, deletes, folder changes, client pairing, and authentication are recorded uniformly across the Native iOS App, iOS Share Extension, Chrome Extension, Raycast, Web Companion, and raw API without per-client tracking code. Events are emitted fire-and-forget — forked and error-swallowed using the same pattern already used to background enrichment — so analytics can never block, slow, or fail a user request, and the Analytics service resolves to a no-op whenever Rybbit is disabled or unconfigured, keeping local development and tests silent by default. Each event identifies the Account only by its opaque better-auth user id as a pseudonymous `user_id`; no email, no original or normalized URLs, and (by default) no host are sent, and event properties are limited to low-cardinality strings and numbers such as Capture Channel, created/updated result, content Type, Folder presence, and Tag count. The v1 event set is `login`, `logout`, `account_created`, `capture`, `item_opened`, `item_deleted`, `item_moved`, `folder_created`, `folder_renamed`, `folder_deleted`, and `client_connected`; enrichment and read-state-toggle events are deliberately deferred as operational rather than usage signals. Rybbit becomes the only analytics tool: the existing Umami browser script is removed from the Web Companion, and Sleevy accepts that this leaves web pageviews and purely on-device client UX (Command Palette, search, Clipboard URL Preview) untracked in v1, preferring one privacy-respecting cross-client product-event stream over partial per-surface web analytics. The Rybbit API key is supplied only through the `RYBBIT_API_KEY` environment variable (Coolify secrets and a gitignored local `.env`) and is never committed; `RYBBIT_API_URL` and `RYBBIT_SITE_ID` are non-secret and may appear in `.env.example`.
